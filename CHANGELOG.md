# Security Policy

## Supported Versions

The following versions of Dance of the Titans currently receive security updates.

| Version | Supported |
|---------|-----------|
| 1.x | ✅ Yes |
| < 1.0 | ❌ No |

---

## Reporting a Vulnerability

The security and integrity of this project are important.

If you discover a potential security vulnerability, please do **not** disclose it publicly through GitHub Issues.

Instead:

1. Contact the project maintainer privately through GitHub.
2. Provide a detailed description of the issue.
3. Include steps to reproduce the vulnerability.
4. If possible, suggest a potential fix.

Reports will be acknowledged as soon as reasonably possible, investigated, and addressed based on their severity.

---

## Scope

This policy applies to:

- JavaScript source code
- Three.js rendering pipeline
- WebGL shaders
- Documentation
- Build and deployment configuration

Third-party libraries are maintained by their respective authors and should be reported through their official channels when appropriate.

---

## Best Practices

Contributors are encouraged to:

- Keep dependencies up to date.
- Validate external input where applicable.
- Avoid exposing sensitive information.
- Test changes before submitting pull requests.
- Follow secure coding practices.

---

## Disclosure Policy

Once a vulnerability has been confirmed and a fix is available:

- A security update will be released.
- The vulnerability will be documented in the project's release notes.
- Contributors who responsibly disclose vulnerabilities may be acknowledged, unless they prefer to remain anonymous.

---

## Contact

For security-related concerns, please use GitHub's private security reporting feature or contact the project maintainer through GitHub.

Thank you for helping keep **Dance of the Titans** secure.
